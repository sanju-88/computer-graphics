Introduction to Computer Graphics

Computer Graphics is a branch of computer science that deals with creating, storing, manipulating, and displaying visual information using computers. It converts data into images, diagrams, charts, animations, and videos to make information visually understandable. Computer Graphics is widely used in engineering, medical science, entertainment, education, business, and research.

Applications of Computer Graphics (Survey)

Computer Aided Design (CAD) :

uses computer systems to assist in the creation, modification, analysis, and optimization of designs.

Uses:

  • Mechanical part design
  • Architectural drawings
  • Electrical circuit design
  • Automobile and aircraft modeling

Advantages:

  • High accuracy and precision
  • Easy modification of designs
  • Reduced design time
  • Better visualization using 3D models

Overview of Graphics Systems

A graphics system consists of input devices, a central processing unit, graphics processing unit, frame buffer, and output devices that work together to generate visual output.
