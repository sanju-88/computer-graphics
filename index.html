Introduction to Computer Graphics

Computer Graphics is a branch of computer science that deals with creating, storing, manipulating and displaying visual content using computers. It converts mathematical data into images and animations for better visualization and understanding.

Applications of Computer Graphics

CAD Modeling

3D Design

Engineering Drawing

Computer Aided Design (CAD)

  • Mechanical part design
  • Architectural blueprints
  • Electrical circuit layout
  • Automobile and aircraft modeling

Overview of Graphics System

A graphics system consists of input devices, CPU, GPU, frame buffer, and output devices working together to generate images on display.

Computer Aided Design

CAD systems allow engineers to design 2D drawings and 3D models with high accuracy, easy modification and efficient simulation.
